# Mini-Diagram

A lightweight, framework-free flowchart builder that runs directly in the browser. Built with Cursor AI in under two hours, this project showcases how simple and powerful vanilla JavaScript can be.

## Features

- **Pure Vanilla JS**: No frameworks, no dependencies - just clean, modern JavaScript
- **Canvas-based Rendering**: Smooth, efficient graphics with HTML5 Canvas
- **Drag & Drop Interface**: Intuitive block creation and positioning
- **Smart Connections**: Connect blocks with auto-routing lines
- **Real-time Editing**: Double-click blocks to add/edit text
- **State Persistence**: Diagrams are saved in the URL for easy sharing

## Technical Highlights

- Modular ES6+ architecture
- Canvas-based text editing system
- LZString compression for URL state
- Event-driven interaction handling
- Efficient render loop with requestAnimationFrame

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start creating diagrams!

That's it! No build process, no package installation, no configuration needed.

## Development

This project was developed using [Cursor](https://cursor.sh), demonstrating how AI-assisted development can produce clean, maintainable code in record time. The entire application, including features like drag-and-drop, connection management, and state persistence, was built in less than two hours.
